Modificar y eliminar tablas

Ejemplos

Agregar una columna

Se agrega la columna "telefono" a la tabla Estudiantes.


            ALTER TABLE Estudiantes
            ADD telefono VARCHAR(15);

Modificar una columna


            ALTER TABLE Estudiantes
            MODIFY telefono VARCHAR(20);

Cambiar nombre de una columna


            ALTER TABLE Estudiantes
            RENAME COLUMN telefono TO celular;

Eliminar una columna


            ALTER TABLE Estudiantes
            DROP COLUMN celular;

Agregar una clave primaria


            ALTER TABLE Estudiantes
            ADD PRIMARY KEY (id);

Agregar una clave foránea

La columna idCarrera de la tabla Estudiantes se relacionará con la columna idCarrera de la tabla Carreras.


            ALTER TABLE Estudiantes
            ADD CONSTRAINT FK_Estudiante_Carrera
            FOREIGN KEY (idCarrera)
            REFERENCES Carreras(idCarrera);

Eliminar una tabla

La siguiente instrucción elimina completamente la tabla Estudiantes.


            DROP TABLE Estudiantes;

Vaciar una tabla

Se eliminan todos los registros de la tabla, pero la tabla sigue existiendo.


            TRUNCATE TABLE Estudiantes;
