Manejo de fechas y horas

Ejemplos

NOW()

Devuelve la fecha y hora actuales del sistema.

SELECT NOW();

Resultado:
2025-06-17 14:30:45

CURDATE()

Devuelve únicamente la fecha actual.

SELECT CURDATE();

Resultado:
2025-06-17

CURTIME()

Devuelve únicamente la hora actual.

SELECT CURTIME();

Resultado:
14:30:45

YEAR()

Obtiene el año de una fecha.

SELECT YEAR('2025-06-17');

Resultado:
2025

MONTH()

Obtiene el número del mes.

SELECT MONTH('2025-06-17');

Resultado:
6

DAY()

Obtiene el día del mes.

SELECT DAY('2025-06-17');

Resultado:
17

HOUR()

Obtiene la hora.

SELECT HOUR('2025-06-17 14:30:45');

Resultado:
14

MINUTE()

Obtiene los minutos.

SELECT MINUTE('2025-06-17 14:30:45');

Resultado:
30

SECOND()

Obtiene los segundos.

SELECT SECOND('2025-06-17 14:30:45');

Resultado:
45

DAYNAME()

Devuelve el nombre del día de la semana.

SELECT DAYNAME('2025-06-17');

Resultado:
Tuesday

MONTHNAME()

Devuelve el nombre del mes.

SELECT MONTHNAME('2025-06-17');

Resultado:
June

DAYOFWEEK()

Devuelve el número del día de la semana.

SELECT DAYOFWEEK('2025-06-17');

Resultado:
3

(En MySQL: 1 = Domingo, 2 = Lunes, ..., 7 = Sábado)

DAYOFYEAR()

Devuelve el número de día dentro del año.

SELECT DAYOFYEAR('2025-06-17');

Resultado:
168

WEEK()

Obtiene la semana del año.

SELECT WEEK('2025-06-17');

Resultado:
24

DATE()

Extrae únicamente la fecha de un valor fecha-hora.

SELECT DATE('2025-06-17 14:30:45');

Resultado:
2025-06-17

TIME()

Extrae únicamente la hora.

SELECT TIME('2025-06-17 14:30:45');

Resultado:
14:30:45

DATE_ADD()

Agrega una cantidad de tiempo a una fecha.

SELECT DATE_ADD('2025-06-17', INTERVAL 10 DAY);

Resultado:
2025-06-27

También se pueden agregar meses o años.

SELECT DATE_ADD('2025-06-17', INTERVAL 2 MONTH);

Resultado:
2025-08-17

DATE_SUB()

Resta una cantidad de tiempo a una fecha.

SELECT DATE_SUB('2025-06-17', INTERVAL 15 DAY);

Resultado:
2025-06-02

DATEDIFF()

Calcula la diferencia en días entre dos fechas.

SELECT DATEDIFF('2025-06-30', '2025-06-17');

Resultado:
13

TIMESTAMPDIFF()

Calcula diferencias en años, meses, días, horas o minutos.

SELECT TIMESTAMPDIFF(YEAR,
'2000-05-10',
'2025-06-17');

Resultado:
25

LAST_DAY()

Obtiene el último día del mes.

SELECT LAST_DAY('2025-06-17');

Resultado:
2025-06-30

EXTRACT()

Extrae una parte específica de una fecha.

SELECT EXTRACT(YEAR FROM '2025-06-17');

Resultado:
2025

SELECT EXTRACT(MONTH FROM '2025-06-17');

Resultado:
6

STR_TO_DATE()

Convierte una cadena de texto en una fecha.

SELECT STR_TO_DATE('17/06/2025','%d/%m/%Y');

Resultado:
2025-06-17

DATE_FORMAT()

Permite mostrar una fecha con un formato personalizado.

SELECT DATE_FORMAT('2025-06-17',
'%d/%m/%Y');

Resultado:
17/06/2025
